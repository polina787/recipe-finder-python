# Recipe Finder (Python)

Python-based application that integrates with the Edamam API to generate recipes based on user-input ingredients and calorie constraints.

## Features
- Filters recipes based on per-serving calorie ranges
- Displays nutritional information (protein, fats, sodium, sugar)
- Allows users to open selected recipes in a web browser

## Technologies
- Python
- REST API (Edamam)
- Requests library

## Overview
Demonstrates API integration, data filtering, and user interaction within a Python application.
